<h1>Footer Navigation Bar</h1>

A simple calculator which allows for basic arithmetic operations. This has been coded using vanilla HTML5 & CSS3 (SCSS).

<strong>Features:</strong><br>
  - Dark themed style
  - Fully responsive
  - Optimised for the web
